# ZooTubeAppProduct
動物のYouTube動画を見れるアプリ
【技術】　Swift5 Xcode12 iOS14 FireBase (Auth Firestore Strage)
【外部ライブラリ】　Alamofire　Nuke　SDWebImage　XLPagerTabStrip　DZNEmptyDataSet　SVProgressHUD
【設計】MVC一部MVVM
【工程】企画設計及び実装まで全て自分で行い、リリースまで行う予定です。
